# How to Install

1. `git clone https://github.com/Solonious/ng-insurance.git`

2. $ `cd ng-insurance`

3. $ `npm install`

4. `ng serve`  (If it does't work try `npm install -g @angular/cli)

5. Navigate to localhost:4200/ or push [Here]('http://localhost:4200/')
